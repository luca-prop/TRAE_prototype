## STEP 0. 서비스 유형 진단

- [ ] **A 유형 (불안 해소형):** 전문가용 서비스, 논문/법률/의료 등
- [ ] **B 유형 (기술 몰입형):** 개발자/디자이너 등 전문가가 직접 쓰는 도구
- [x] **C 유형 (결과 지향형):** 복잡한 과정을 줄여주는 자동화 솔루션

> **선택 근거**: 씨드핏은 "주 15시간 걸리던 재개발 구역 탐색"을 "가용 현금 입력 한 번으로 즉시 결과 도출"해주는 자동화 분석 플랫폼. 사용자(스마트 비기너)는 복잡한 내부 로직을 알 필요 없이, "입력 → 결과"의 단순한 경험을 원하는 결과 지향형 고객.

## STEP 1. [공통] 핵심 코어 필수요소 점검

> **목표: 6개 모두 체크**

### ① 히어로 섹션 (Hero Section) — 첫인상 3초

| 항목 | 체크 | 구현 내용 |
|------|:----:|-----------|
| **명확한 헤드라인** | ✅ | `"가용 현금만 입력하면 타겟 구역이 즉시 나옵니다"` — 고객의 최종 이득(즉시 결과)을 한 문장으로 요약 |
| **보조 설명 (Sub-head)** | ✅ | `"씨드핏이 3초 안에 내 예산으로 진입 가능한 재개발 구역을 찾아드립니다"` — 구체적 방법(3초/예산 기반)을 보충 설명 |
| **직관적 이미지** | ⚠️ | 그래디언트 글로우 효과 + 다크 배경으로 시각적 분위기 연출. 실제 서비스 스크린샷/영상 대신 텍스트 중심 구성 (향후 데모 영상 추가 권장) |

### ② 행동 유도 (CTA) — 전환 장치

| 항목 | 체크 | 구현 내용 |
|------|:----:|-----------|
| **동사형 버튼** | ✅ | `"내 예산으로 구역 찾기"`, `"무료로 분석 시작하기"`, `"지금 무료로 시작하기"` — 모두 동사형 행동 유도 |
| **반복 배치** | ✅ | **상단**(Nav CTA) + **히어로**(메인 CTA) + **중단**(Mid CTA 섹션) + **하단**(Final CTA 섹션) = **총 4회 반복 노출** |

### ③ 신뢰의 증거 (Social Proof) — 의심 제거

| 항목 | 체크 | 구현 내용 |
|------|:----:|-----------|
| **숫자/로고 증명** | ✅ | 수치 증명: `1,247개 분석 대상 구역`, `15시간→30초 탐색 시간`, `±5% 오차 이내`, `0% 허위매물` / 로고 월: `국토교통부`, `한국부동산원`, `서울시 정비사업`, `공인중개사협회` |

## STEP 2. [선택] 유형별 전략 요소 점검

> **핵심 목표: "편의성"과 "결과물"**

### 🅲 C 유형 선택 시 (Macaly 전략)

| 항목 | 체크 | 구현 내용 |
|------|:----:|-----------|
| **[Input-Output]** 입력→결과 단순 구조 | ✅ | **"HOW IT WORKS" 섹션**: `STEP 1 가용 현금 입력` → `STEP 2 씨드핏 엔진 분석` → `STEP 3 맞춤 구역 리스트` 3단계 다이어그램으로 복잡한 역산 엔진을 블랙박스화 |
| **[결과 갤러리]** 완성도 높은 결과물 예시 | ✅ | **"RESULT PREVIEW" 섹션**: `노량진 1구역(95%)`, `흑석 9구역(72%)`, `수색 3구역(65%)` — 실제 분석 결과 카드 3건을 프리뷰로 제시. 매칭 적합도 프로그레스 바 포함 |
| **[Before & After]** 도입 전후 효과 비교 | ✅ | **"BEFORE & AFTER" 섹션**: 5개 항목(탐색 시간, 비교 분석, 매물 신뢰도, 분담금 오차, LTV 반영)에 대해 기존 방식 vs 씨드핏 대비표 제공 |

## STEP 3. [최종] 자가 피드백 (Self-Feedback)

> ✅ **"내 예산만 넣으면 바로 결과가 나오는구나. 한번 써볼까?"**

- Before & After 테이블이 기존 방식의 고통을 상기시키며 전환 동기를 자극
